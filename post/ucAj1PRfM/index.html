在Excel中根据单元格填充颜色计数

2019-10-23 Excel ， Visual Basic

在维护我的城市轨道交通乘降记录表时，我使用了不同的填充颜色来标记我在某个站点乘降时是否进出了付费区。因此在统计某条线路的乘降比时，我希望能够直接按单元格填充颜色来计数。经过检索后发现达成这个目的主要有利用GET.CELL函数定义名称和使用VBA编写函数两种方式，而我认为后者的使用更加方便灵活。

如果只需要将某个单元格的颜色映射到数值，可以定义如下的VBA函数

Function GetColor(range_data As Range) As Long
    GetColor = range_data.Interior.ColorIndex
End Function

这个函数将返回输入单元格的ColorIndex，例如单元格A1的填充颜色为黑色，则GetColor(A1)的值即为1。值得注意的是ColorIndex是按照默认调色盘的颜色顺序定义的，一共只有56个值，但是这对我的需求来说已经绰绰有余了。

如果需要对某个区域内特定填充颜色的单元格计数，则可以传入一个ColorIndex作为参数

Function CountColor(range_data As Range, color_index As Long) As Long
    Dim data As Range
    For Each data In range_data
    	If data.Interior.ColorIndex = color_index Then
        	CountColor = CountColor + 1
    	End If
	Next data
End Function

实际操作中查询ColorIndex表并不方便，可能还需要额外调用一次前述的GetColor函数，因此不如将其集成到函数中

Function CountColor(range_data As Range, criteria As Long) As Long
    Dim data As Range
    Dim color_index As Long
    color_index = criteria.Interior.ColorIndex
    For Each data In range_data
    	If data.Interior.ColorIndex = color_index Then
        	CountColor = CountColor + 1
    	End If
	Next data
End Function

这样调用函数CountColor(A1:A10,B1)即可返回单元格A1-A10中填充颜色和单元格B1一致的数目。
