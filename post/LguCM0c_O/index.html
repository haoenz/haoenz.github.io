在PowerShell中使用conda

2021-12-05 Python ， PowerShell

最新版本的Anaconda其实自带PowerShell Prompt了，只是使用的还是Windows 10的默认终端conhost.exe。似乎Windows 11的默认终端已经更新成Windows Terminal了，但是我暂时还不想升级到Windows 11，因此需要做一些非常简单配置。

其实只有一条命令：打开Anaconda Prompt，运行

conda init powershell

这样不论从哪里打开PowerShell，都能够自动载入Anaconda的环境了。不过部分oh-my-posh的主题可能无法正常显示conda所处的环境，因此可能还需要在主题文件中加入一项：

{
     "type": "envvar",
     "style": "plain",
     "foreground": "#FFDF59",
     "properties": {
         "var_name": "CONDA_DEFAULT_ENV"
         }
}
