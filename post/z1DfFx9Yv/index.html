UCAS雁栖湖花式进/出城之二-九

2021-04-28 轨道交通 ， 轨迹记录

本文撰写于2021年2月末，最初发表在知乎专栏上，在这里重发一方面是为了整理自己写过的东西，另一方面也是为了测试如何在Hexo中插入图片题注。

关于路程时间：候车时间不计（因此每个方案的总用时仅供参考），室内换乘时间可能有一定误差，数据参考高德地图&盛名时刻表。路径中铁路部分具体用时以我首次使用该路径时乘坐的车次为准。

关于地铁票价：即使在地铁网络中并未按最短路径运行，也依然按最短路径计算票价（这也是实际中的情况），不考虑超时乘坐的额外收费，不考虑在路径中途进出收费区产生的额外费用。

关于公交票价：按使用市政公交卡5折优惠处理。

UCAS雁栖湖花式进/出城之二

起点：北京南站

终点：UCAS雁栖湖校区

路径：

  1. 北京南站---[地铁4号线]-->宣武门---[地铁2号线（外环）]-->北京站，票价4元，用时约23分钟；
  2. 北京站--[K1189次]-->怀柔北站，票价12.5元，用时约1小时28分钟；
  3. 怀柔北站---[步行]-->UCAS东二门，里程1.2km，用时约17分钟。

总用时约2小时8分钟，费用16.5元。

K1189次是目前唯一一班从北京站出发经停怀柔北站的火车。因此如果时间合适，本方案可能是北京南站或者北京站到达的同学前往UCAS雁栖湖最快的公共交通方式。

UCAS雁栖湖花式进/出城之三

起点：UCAS雁栖湖校区

终点：北京南站

路径：

  1. UCAS东二门---[步行]-->怀柔北站，里程1.2km，用时约17分钟；
  2. 怀柔北站---[市郊铁路怀密线]--->北京北站，票价9元，用时约1小时37分钟；
  3. 西直门---[地铁4号线]-->北京南站，票价4元，用时约22分钟。

总用时约2小时16分钟，费用13元。

本方案在大部分时间段UCAS雁栖湖进/出城的最快的公共交通方式之一，目的/出发地在西城区、海淀区时一般为首选方案。

UCAS雁栖湖花式进/出城之四

起点：UCAS雁栖湖校区

终点：北京站

路径：

  1. UCAS东二门---[步行]-->怀柔北站，里程1.2km，用时约17分钟；
  2. 怀柔北站---[K1190次]-->顺义站，票价9元，用时约43分钟；
  3. 顺义站---[步行]-->顺义（地铁站），里程1.4km，用时约19分钟；
  4. 顺义---[地铁15号线]-->望京西---[地铁13号线]-->西直门---[地铁2号线（内环）]-->北京站，票价7元，用时约2小时8分钟。

总用时约3小时27分钟，费用16元。

本方案不适合日常通勤使用。

K1190次原本可以直接抵达北京站，绕路的顺义的目的是体验15号线。抵达望京西后绕路13号线前往西直门的目的是和五道口的同学约饭（产生的额外轨迹途中并未绘出）。西直门和北京站正好位于2号线的西北角和东南角，但是实际上从西直门站乘坐2号线前往北京站外环方向比内环方向更快，之所以绕路内环方向是因为……闲的（x。

UCAS雁栖湖花式进/出城之五

起点：北京站

终点：UCAS雁栖湖校区

路径：

  1. 北京站---[地铁2号线（外环）]-->东直门，票价3元，用时约22分钟；
  2. 东直门枢纽站---[916路快车]-->明珠广场，票价5.5元，用时约1小时6分钟；
  3. 明珠广场（公交站）---[步行]-->怀柔府前街（公交站），里程518m，用时约7分钟；
  4. 怀柔府前街---[H58路]-->怀北庄，票价1.5元，用时约51分钟；
  5. 怀北庄（公交站）---[步行]-->UCAS东二门，里程230m，用时约4分钟。

总用时约2小时30分钟，费用10元。

公交站点信息更新：明珠广场已于2021年2月6日更名为怀柔区中医院。

916快终于登场了！出城方向于东直门枢纽站首班5:50发车，末班20:00发车，高峰期发车间隔仅有几分钟。于北三环太阳宫桥上京承高速，至怀柔桥下高速。在交通顺畅的前提下从东直门至怀柔城区用时仅略多于一小时，是怀柔城区居民快速进入北京市核心区域的首选。对于雁栖湖的UCASer来说，在因为出发时刻、出发/目的地方位等因素不方便乘坐火车的情况下，也是较好的进/出城选项之一，而且更便宜（如果办了公交学生卡还能再便宜点）。

缺点在于：

  • 需要前往怀柔城区换乘
  • 有堵车风险

916快在怀柔城区停靠站点较多，因此前往雁栖湖的换乘方案也多种多样，本方案在府前街换乘是为了……在青春万达斜对面的麦当劳吃早饭。

UCAS雁栖湖花式进/出城之六

起点：UCAS雁栖湖校区

终点：北京南站

路径：

  1. UCAS东二门---[步行]-->怀北庄（公交站），里程230m，用时约4分钟；
  2. 怀北庄---[H64路]-->兴怀大街中路，票价1.5元，用时约52分钟；
  3. 兴怀大街中路（公交站）---[步行]-->后横街（公交站），里程496m，用时约7分钟；
  4. 后横街---[916路快车]-->太阳宫桥，票价5.5元，用时约1小时13分钟；
  5. 太阳宫桥（公交站）---[步行]-->光熙门（地铁站），里程149m，用时约2分钟；
  6. 光熙门---[地铁13号线]-->东直门---[地铁2号线（内环）]-->宣武门---[地铁4号线]-->北京南站，票价5元，用时约38分钟。

总用时约2小时56分钟，费用12元。

公交站点信息更新：兴怀大街中路已于2021年2月6日更名为兴怀大街。

本方案基本上是方案五的反向，与方案五类似，本方案在后横街换乘是为了……在普法公园北边的另一家麦当劳吃午饭。不过初次使用本方案时得到的教训是：由于916快在怀柔城区的运行路径近乎为一个狭长的长方形，因此在高峰时期可以考虑提前几站下车/推后几站上车，步行穿越这个长方形的短边，说不定比堵在路上更快些。

初次使用本方案时还遇到了京承高速进京方向的轻微堵车，因此为了弥补堵车的时间提前在太阳宫桥下车换乘地铁了。

UCAS雁栖湖花式进/出城之七

起点：UCAS雁栖湖校区

终点：清河站

路径：

  1. UCAS东二门---[步行]-->中国科学院大学东区（公交站），里程740m，用时约10分钟；
  2. 中国科学院大学东区---[快速直达专线197路]-->地铁望京西站，票价12.5元，用时约1小时9分钟；
  3. 地铁望京西站（公交站）---[步行]-->望京西（地铁站），里程189m，用时约3分钟；
  4. 望京西---[地铁13号线]-->芍药居---[地铁10号线]-->知春路---[地铁13号线]-->清河，票价6元，用时约37分钟。

总用时约1小时59分钟，费用18.5元。

觉得916快还要进怀柔城区换乘太麻烦？快专197可以解决这个问题！东区南门口（因疫情管控暂时封闭中）直接上车，绕开怀柔城区，走省道S227从杨雁桥上京承高速，至北四环北湖光桥下高速后立即到达终点站。全程国道/省道/高速，沿途各站点若无乘客乘降则直接不减速越行，在交通顺畅的前提下可能对于大部分北京城六区的目的/出发地来说都比市郊铁路怀密线还要快。

缺点在于：

  • 依然有堵车风险，我初次使用此方案时就因为京承高速上一辆侧翻的冷藏车而堵了一个多小时
  • 班次太少且时刻不太合适，进城方向6:00/7:00/16:10/16:50发车，出城方向8:00/9:00/18:10/18:50发车
  • 票价稍贵一些

本方案中从望京西前往清河并未直接乘坐13号线是因为我要去太阳宫修手机（产生的额外轨迹图中并未绘出）。

UCAS雁栖湖花式进/出城之八

起点：清河站

终点：UCAS雁栖湖校区

路径：

  1. 清河---[地铁13号线]-->光熙门，票价6元，用时约38分钟；
  2. 光熙门（地铁站）---[步行]-->西坝河（公交站），里程861m，用时约12分钟；
  3. 西坝河---[916路快车]-->庙城东，票价5元，用时约43分钟；
  4. 庙城东（公交站）---[步行]-->庙城东（公交站），里程275m，用时约4分钟；
  5. 庙城东---[H58路]-->怀北庄，票价2元，用时约1小时7分钟；
  6. 怀北庄（公交站）---[步行]-->UCAS东二门，里程230m，用时约4分钟。

总用时约2小时48分钟，费用13元。

本方案是……漏乘了市郊铁路怀密线末班车（18:16发车）后的无奈之举（为什么市郊铁路非要让国铁来运营搞什么3分钟停止检票啊喂！），我在这一个晚上连续犯傻多次，相应地也收获了很多经验：

  • 不要把市郊铁路当地铁一样赶（太蠢了
  • 北京市有很多同名但相差十几至几十米的公交站点，注意分辨（哪一个西坝河？哪一个庙城东？）
  • 相比于916快，866在北京城区始发站更靠北、出城末班车更晚、在怀柔城区运行的路径更短（上了京承高速看到旁边开过一辆866才反应过来刚才在望京西下就行了不用坐到光熙门的）
  • H58末班车庙城东20:25发车，约21:30抵达怀北庄，是从怀柔城区前往UCAS雁栖湖最晚的一班公共交通工具

UCAS雁栖湖花式进/出城之九

起点：UCAS雁栖湖校区

终点：清河站

路径：

  1. UCAS东二门--[打车]-->怀柔汽车站，费用约30元，用时约30分钟；
  2. 怀柔汽车站--[866路]-->地铁望京西站，票价5元，用时约1小时；
  3. 地铁望京西站（公交站）---[步行]-->望京西（地铁站），里程189m，用时约3分钟；
  4. 望京西--[地铁13号线]-->清河，票价6元，用时约31分钟。

总用时约2小时4分钟，费用41元。

本方案是错过了S514次（16:38发车）后想到的进城方案，虽然最后发现不论在时间上还是费用上都不如老实等待下一班的S504次（18:30发车）……就当是体验了一次866吧。

最后放一张2020年下半年作为一名雁栖湖里的咸鱼使用过的进出城路径总结：

可以看出怀柔城区与北京城区的主要交通线路为：京通铁路、京承高速、京承铁路。对应的进/出北京城六区主要方式分别为：

  • 京通铁路：市郊铁路怀密线
  • 京承高速：866路、916路快车、快速直达专线197路，UCAS校车
  • 京承铁路：市郊铁路通密线、K1189/90次、K7711/2次、6419/20次

其中斜体部分为我还没有乘坐过的车次/线路，除此之外，还有一些其他线路（如京沈高铁、国道G101等）也尚未体验过，希望2021年能有机会尝试乘坐吧。
